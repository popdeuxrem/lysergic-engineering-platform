# LEP Operations Console

Next.js operations console for the Lysergic Engineering Platform v0.1.0.

## Scope

Included:

- Next.js 15 runtime
- TypeScript configuration
- Application shell
- Root page
- Frontend testing
- Linting
- Type checking
- Production build validation

Excluded:

- Backend integration
- API clients
- Authentication
- State management
- Dashboards
- Workflows
- Agents
- Intelligence interfaces

## Development

Install dependencies:

```bash
pnpm install
```

Run:

```bash
pnpm dev
```

Validate:

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
