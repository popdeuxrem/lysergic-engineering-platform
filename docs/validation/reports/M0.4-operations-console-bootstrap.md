## Contract Deviations

1. **Next.js version**: Contract specified "Next.js runtime" without version constraint. Next.js 16.2.10 was selected as the latest stable release to avoid known security vulnerabilities in 15.x versions (CVE-2025-66478).
2. **Dockerfile install strategy**: `pnpm install --no-frozen-lockfile --ignore-scripts` used because the pnpm workspace lockfile is at repository root (outside Docker build context). Build scripts ignored to prevent pnpm 10 build approval errors in non-interactive Docker builds.
3. **Root pnpm-lock.yaml**: Generated at repository root as a side effect of workspace dependency installation. Included in commit for reproducible builds.
